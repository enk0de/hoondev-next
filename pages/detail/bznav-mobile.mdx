## 주요작업 요약

- Echarts를 이용해 차트 구현 및 데이터 시각화
- 유저들에게 사업 인사이트를 주기적으로 발행하여 제공해주는 **인사이트 리포트 서비스** 전체 개발
  - 종합소득세 세무진단 리포트, 예상부가세 리포트, 연결산 리포트 등
- **모바일 거래처 관리 기능** 전체 개발
- **홈택스 아이디/비밀번호 찾기 및 재설정 기능** 전체 개발
- **주민등록번호 입력 플로우** 구현
- **사업용신용카드 공제/불공제 증빙** 구현
- 인사이트 리포트 유저 피드백 기능 구현
- 레거시 증빙 쿼리를 새로운 인터페이스로 만들어진 쿼리로 반영하기 위한 대대적인 마이그레이션 작업
  - 신규 인터페이스 기반 데이터 추상화, 인터페이스 모델링
  - 신규 모델링 기반의 신규 컴포넌트 및 페이지 구현
  - 백엔드 모델링 문제로 인해 프론트엔드 단에서 통신 데이터를 2차 가공하기 위한 파이프, 유틸 함수 등을 구현
- 주민등록번호 해시 암호화로 인한 프론트엔드 모델링 및 로직 수정 작업
- 더 나은 UX 플로우를 위해 디자이너/기획자와 함께 커뮤니케이션하며 적극적으로 개선
