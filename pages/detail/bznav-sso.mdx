## 주요작업 요약

- UI 라이브러리를 사용하지 않고 Pure하게 디자인 컴포넌트 제작
- 로그인 뷰 구현
- 휴대폰 인증번호 전송 및 검증 기능 구현
- 회원가입 뷰 구현
  - 비동기 이메일 중복 검사 기능
  - 폼 유효성 검사 기능
  - 서비스 약관 동의 및 약관 상세 모달
- 비밀번호 재설정 기능 및 뷰 구현
- 통신 로직과 뷰 로직 분리하는 리팩토링 작업
- Google Analytics 매트릭 로깅
