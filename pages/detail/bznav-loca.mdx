## 주요작업 요약

- Webpack 5 Module Federation을 이용하여 **마이크로 프론트엔드** 아키텍쳐 설계
- 인사이트 리포트 앱을 Federation하여 **호스트 앱 배포 없이** 비즈니스 로직이 반영되도록 구현
- Federated되는 모듈 사이의 의존성 패키지 vendor 공유
- **Apollo Client**를 이용해 GraphQL 통신 구현
- **Apollo Link**를 이용하여 글로벌 에러 핸들링, 동적 엔드포인트/Headers 구현
- **GraphQL Generator**를 이용하여 Apollo Client 기반 React Hooks, 정적 타입 생성
- QA 및 CS 처리를 위한 개발자 모드 구현
- Yarn 3를 이용한 pnp, Zero-install 적용
- Custom Webpack 설정으로 최신 기능 사용 및 번들 사이즈 최적화
- HMR, react-refresh 적용
- emotion, TailwindCSS 사용하여 스타일 환경 구성
- Yarn Workspace를 이용한 모노레포 구성
- ESLint, Prettier를 이용한 코드 포맷 획일화
