## 주요작업 요약

- **프론트엔드 개발 책임**으로서 프로젝트를 리딩했습니다.
- 기본 프로젝트 **골격**을 설계하고 **코딩 컨벤션**을 확립하였습니다.
- 비동기적인 데이터를 효율적으로 제어하기 위해 **RxJS**를 사용하였습니다.
- WebCam과 Tensorflow.js 머신러닝 모델의 연동을 위한 비즈니스 로직을 구현하였습니다.
- Socket.IO를 이용해 스터디 룸에서 **실시간으로 공부 데이터를 공유**하는 기능을 구현하였습니다.
- 데이터 시각화와 후처리를 위해 **서버에 비동기적으로 학습 데이터를 가공하여 전송**하는 기능을 구현하였습니다.
- 실험적으로 Atomic Design Pattern을 디자인 시스템 컴포넌트를 구현하는데 도입하였습니다.
- **OAuth 인증 처리**를 위한 플로우와 인증 데이터 기반으로 **라우트의 접근 스코프를 제한**하는 컴포넌트를 구현하였습니다.
- Antd Chart를 이용하여 **학습 데이터를 시각화**하는 기능을 구현하였습니다.
- ESLint, Prettier 등을 이용하여 코드 포맷을 획일화하였습니다.
