## 작업 요약

- 서비스 기획, 아키텍쳐 설계, 인프라 구축 등 개발 전두지휘
- UI/UX 기획 및 디자인
- 비동기 통신 로직의 제어를 위해 RxJS 사용
- OAuth 인증 처리를 위한 플로우, 접근 스코프 구현
- emotion, TailwindCSS 사용하여 스타일 독립화
- Antd Chart를 이용하여 데이터 시각화
- ESLint, Prettier 이용하여 코드 포맷 획일화
- 통신 독립적인 뷰를 그리는 컴포넌트, 통신을 위한 컴포넌트 구분하여 통신 독립화
- 직접 UI/UX 고민하고 디자인
- Heroku, Vercel을 이용하여 서비스 배포
- 교내 해커톤에서 1위 수상

## 기술 스택

### 코어

- React 17
- TypeScript
- RxJS for Reactive Programming
- OAuth 2.0 for Access-Token handling
- Context API
- React Router
- Axios for using RESTful API
- Local Storage
- Craco for customizing CRA

### 스타일링

- @emotion
- TailwindCSS
- twin.macro

### 데이터 시각화

- Antd
- Antd Chart

### 코드 린팅

- ESLint
- Prettier
- @emotion/babel-preset-css-prop

### CD

- Vercel
- Heroku
