## 작업 요약

- 프론트엔드 아키텍쳐 설계, 코딩 컨벤션 확립 등 챕터 전두지휘
- 비동기 데이터 흐름을 제어하기 위해 RxJS 사용
- WebCam과 Tensorflow.js 머신러닝 모델 연동을 위한 로직 구현
- Socket.IO를 이용해 스터디 룸에서 실시간 통신 기능 구현
- 데이터 시각화와 후처리를 위해 서버에 비동기적으로 학습 데이터 전송하는 로직 구현
- 아토믹 디자인 패턴을 차용하여 디자인 시스템 컴포넌트 구현 (실험적)
- OAuth 인증 처리를 위한 플로우, 접근 스코프 구현
- emotion, TailwindCSS 사용하여 스타일 독립화
- Antd Chart를 이용하여 데이터 시각화
- ESLint, Prettier 이용하여 코드 포맷 획일화
- 통신 독립적인 뷰를 그리는 컴포넌트, 통신을 위한 컴포넌트 구분하여 통신 독립화
- Heroku, Vercel을 이용하여 서비스 배포
